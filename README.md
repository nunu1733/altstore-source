# altstore-source

SideStoreとAltStore Classic向けのアプリ配布元です。
現在掲載しているアプリはNarouReader（「小説家になろう」専用リーダーアプリ）です。

## Source URL

SideStoreとAltStore Classicには次のURLを登録します。

```
https://altstore.xnunux.com/source.json
```

## SideStore

SideStoreの公式インストール案内に従ってSideStoreを準備してください。

- SideStoreのインストール: https://docs.sidestore.io/docs/installation/install
- Sourceの追加手順: https://docs.sidestore.io/docs/advanced/app-sources

## AltStore Classic

このSourceはAltStore Classic向けです。
AltStore PALでは利用できません。

### macOS

AltStore Classicの公式macOS向けインストール案内に従ってAltServerを準備してください。

- AltStore Classic（macOS）のインストール: https://faq.altstore.io/altstore-classic/how-to-install-altstore-macos

### Windows

AltStore Classicの公式Windows向けインストール案内に従ってAltServerを準備してください。

- AltStore Classic（Windows）のインストール: https://faq.altstore.io/altstore-classic/how-to-install-altstore-windows

### Sourceの追加

AltStore Classicを起動し、Sourcesタブの追加ボタンから Source URL を登録します。
AltStore Classicの基本的な使い方は公式Getting Started案内を参照してください。

- AltStore Classic Getting Started: https://faq.altstore.io/altstore-classic/your-altstore

## NarouReader

iOS向けの「小説家になろう」専用リーダーアプリです。
本文テキストをAPIで取得せず、広告除去を目的とした通信ブロックを行いません。
API取得データはメタデータのみとし、小説家になろう公式規約に従います。
